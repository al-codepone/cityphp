Directory Structure

The cityphp directory contains the entire framework. You must upload the cityphp directory to your web server. You can place the cityphp directory anywhere on your web server. Each PHP script that uses the framework must define a CITY_PHP constant that points to the cityphp directory using an absolute path. For example:

<?php

define('CITY_PHP', '/home/user/cityphp/');

?>
