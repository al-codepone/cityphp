Introduction

cityphp is a PHP 5 web application framework. The framework features a template system, database system, form handler and paginator. The framework does not come with any built-in HTML, CSS, JavaScript or database tables; you must build and integrate these things using the framework. The framework has no dependencies. The framework itself is predominantly object oriented. When using the framework you write both object oriented and procedural code.

You use the template system to render structured documents like HTML documents. The template system is based on object composition and an object interface with one method.

The database system supports most PHP RDBMS vendors including MySQL and PostgreSQL. When you use the database system you must write all the raw vendor specific SQL.

The form handler validates and collects POST variables. The paginator computes the total number of pages and the current page number.
