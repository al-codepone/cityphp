Paginator

To use the Paginator class you pass in the number of items, items per page and the current page number. Then you can get the normalized current page number and the number of pages. Here's an example:

<?php

require_once('./constants.php');
require_once(CITY_PHP . 'pagination/Paginator.php');

$numItems = 125;
$itemsPerPage = 20;
$currentPageNum = 2;
$paginator = new Paginator($numItems,
    $itemsPerPage,
    $currentPageNum);

printf('page %d of %d', $paginator->getCurrentPageNum(),
    $paginator->getNumPages());

?>
